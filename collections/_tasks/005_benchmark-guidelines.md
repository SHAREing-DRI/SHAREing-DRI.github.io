## Fit to programme

This task has been identified by the working groups as part of the agenda behind [WP 1.2](/about/workinggroups).

The task number is 005.


## Description

SHAREing wants to work towards a generic and simple performance assessment service.
For this, the project has to curate some guidelines.
The objective of this task is to kickstart the curation of a performance assessment hitchhiker guide that SHAREing can continue to befill throughout the project's lifetime.


## Outcomes

- Start a guidebook.
- Publish first versions on SHAREing webpage.

This project has now finished. The outcomes can be found on the [SHAREing webpage](/performance-assessment/guidebook), which is a living document.
