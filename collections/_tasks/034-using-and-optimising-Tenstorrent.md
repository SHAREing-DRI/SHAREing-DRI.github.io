## Description
The HPC Hardware Lab at Durham has recently started hosting a Tenstorrent BlackHole system which offers impressive AI inference performance, and is likely suitable for HPC.  We invite proposals to make use of this system, explore its performance, document how to use it, and develop experience in compiling and porting codes.

This is an open call, and multiple responses may be accepted if they focus on different aspects of the system.

More information about the system is given here: https://cosma.readthedocs.io/en/latest/tenstorrent.html

## Approach

Proposals should focus on how they intend to use the system, what benchmarks they will deliver, what codes will be used, and details around code porting.


## Outputs

Documentation for the SHAREing and testbed websites, including user guides and performance results.
